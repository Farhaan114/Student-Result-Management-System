# Student-Result-Management-System
Done using ReactJS, NodeJS, ExpressJs, MySQL

A system where students can view their exam results and a platform for the University admins to add/remove/remove students or results.
The project aims at establishing a robust system for the administration of results in a University.

## Pages-involved: 
  - View Results page
   - <picture></picture>
  - Admin login page
  - Admin Home
  - Manage Students
  - Manage Results
  - Add/edit a student 
  - Add a result

## Features of the tech-stack used in the project:
  - React.js: Hooks, States and props
  - REST API requests (designed with POSTMAN)
  - BackEnd: Node.js and express.js
  - Axios: used for making requests between the end-points
  - Basic CSS: used for styling, etc. 

## Conclusion
**The overall functionality of the project ensures efficient administration of student results. Deployment of such a system can allow on-demand result fetching and easy and efficient managing of results by the admins.**
